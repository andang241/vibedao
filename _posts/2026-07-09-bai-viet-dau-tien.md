Đây là bài viết mẫu để bạn thấy blog hoạt động như thế nào.

## Cách viết bài mới

Mỗi bài viết là một file Markdown nằm trong thư mục `_posts/`, đặt tên theo
định dạng:

```
NĂM-THÁNG-NGÀY-tieu-de-bai-viet.md
```

Ví dụ: `2026-07-09-bai-viet-dau-tien.md`.

Đầu mỗi file có một khối cấu hình gọi là **front matter** nằm giữa hai dấu `---`:

```yaml
---
layout: post
title: "Tiêu đề bài viết"
date: 2026-07-09 10:00:00 +0700
categories: [ghi-chú]
tags: [jekyll, github-pages]
---
```

## Định dạng nội dung

Bạn viết nội dung bằng Markdown bình thường:

- **In đậm**, *in nghiêng*
- Danh sách, bảng, ảnh
- Chèn code:

```python
print("Xin chào GitHub Pages!")
```

> Sau khi lưu và push lên GitHub, bài viết sẽ tự động xuất hiện trên trang chủ.

Chúc bạn viết blog vui vẻ!
